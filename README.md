# Library Management System (Java Console Application)

## Project Overview

This project is a **console-based Library Management System developed using Java**.
It allows an administrator to manage books in the library and students to register, borrow, and return books.

The system demonstrates **Object-Oriented Programming (OOP), ArrayList usage, and basic library operations**.

---

## Features

### Admin Module

* Admin login authentication
* Add new books to the library
* View all books in the library
* View issued books

### Student Module

* Student registration (Sign Up)
* Student login (Sign In)
* View available books
* Borrow books
* Return borrowed books

---

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner (User Input)
* Basic File Handling

---

## System Workflow

### Admin

1. Login with Admin ID and Password
2. Add books to the library
3. View available books
4. Check issued books

### Student

1. Register with name and password
2. Login using student ID and password
3. View available books
4. Borrow a book
5. Return the borrowed book

---

## How to Run the Project

### Step 1: Compile the Program

```bash
javac Main.java
```

### Step 2: Run the Program

```bash
java Main
```
